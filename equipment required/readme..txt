This readme file details the additional equipment required to run VCT experiments.

The equipment consists of:

1. a stand: Used to position the cone board at 45 degrees and allow the user host arm to stimulate the mosquitoes from under the board.
2. two acrylic sheets: A3 in size with holes for the WHO cone and holes to mount the smartphone holder. 
3. other equipment (phone holder and angle adjuster): This can be found in the file 'other equipment link.txt'  The suggested links are to items on amazon uk. However alternative options should be available at many other online shopping stores.
4. Smartphone. VCT experiments have been conducted with iphoneSE v2, however any smartphone capable of video recording should suffice. The required resolution can be less than 720p, framerate of only 30fps. The author recommends disabling software and hardware enhanced recordings such has HEVC, HDR, and smart AI features. A simple .mp4 or .mov file is preferred for analysis. If a video file is flipped or slightly rotated this can be corrected when running the ViCTA cone test analysis, although the rate of processing will be reduced.


3D Printing files:

cone test stand - side panel.stl     (x2)

cone test stand - connecting rod.stl (x3)

These files can be used to direct a 3D printer to make the parts for the cone test stand.

The three rods fit between the two sides. The ends of the rods are designed to have a very tight fit so some adjustment with a manual file may be required.


Cone test Sheets:

These should be 2x A3 (approx 420x300mm) acrylic sheets of 3mm thickness for each sheet. There should be a 9cm diameter hole close to the bottom of the sheet to allow a human host arm to stimulate the presence of mosquitoes.

A series of smaller 6mm holes should be made through the centre of the sheets to mount the mobile phone holder.

An approximate illustration of the desing for the A3 sheets is shown in an image file and attached as a dxf file. When drilling the holes in the sheets it is recommended to tape two sheets together before drilling the holes, to ensure correct alignment.

The arrangement of testing net sample and the cone and the boards is shown in file 'cone layers'. The acrylic boards are denoted with a blue colour. The cone sits on top of the net sample and both cone and net sample are between the acrylic sheets.

The bottom layer should be a piece of white coloured embroidery fabric. this is used to occlude the host arm from the recording.
