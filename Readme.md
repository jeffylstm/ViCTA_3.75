Readme file for: Video Cone Test Analysis (ViCTA)

This repository stores the required files to run VCT (Video Cone Tests) an augmentation of the standard WHO cone bioassay to monitor the amount of mosquito activity.

The VCT system requires only simple equipment (3D printed stand, modified acrylic cone boards with a hole for the cone so that the host attractant arm can be sensed, and a basic smartphone).

WHO cone tests are recorded for their 3m duration with a host arm underneath to provide attractant stimuli.

The recordings can later be analysed using the ViCTA software to assess the amount of motion in the cone. The output of the ViCTA software can be pasted into an excel data entry template.

When all replicates of different mosquito strains and different net treatments have been completed, R scripts are provided to extract the data into a central file format and R scripts are provided for simple preliminary visualisation.
