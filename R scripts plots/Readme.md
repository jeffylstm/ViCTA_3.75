Readme file for: R scripts plots

This folder contains simple R scripts to read from the ViCTA centralised data file and generate plots for preliminary data analysis


The source data, from a series of VCT experiments with Anopheles funestus FANG and FUMOZ colonies in the extracted centralised file format, can be found in the Data folder in a file called JessieFangFumozVCD.xlsx'. Output plots generated from the scripts can be found in the 'Results/Jessie folder as .png format plots.'

to load the project into R studio, double click on the victa_allsites.Rproj file. The user may need to preinstall the required R packages before the script will run. The central file database is loaded and parsed in the file 'JessieFangFumoz.R' and this file imports the source code of 'PlotUtils.R', some of which is used to generate a selection of example results plots.
