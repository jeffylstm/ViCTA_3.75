Readme file for: R scripts plots

This folder contains simple R scripts to read from the ViCTA centralised data file and generate plots for preliminary data analysis


The source data, from a series of VCT experiments with Anopheles funestus FANG and FUMOZ colonies in the extracted centralised file format, can be found in the Data folder in a file called JessieFangFumozVCD.xlsx'. Output plots generated from the scripts can be found in the 'Results/Jessie folder'
